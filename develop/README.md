# Professional README Generator

## Description

While creating projects on Github, It is necessary for users to have a professional README in order for others to understand the purpose of the project, how the application was developed, how to properly use the application, and how to contribute to the project. This project allows users to create a professional README using a command-line application. This application will help users develop a professional README with less time.

## Installation

Download the command-line application.

## Usage

In order to use the command-line application, the user will download the application, then after running the application, open the terminal and type the words "node index.js". Afterwards, the user will answer the following questions in order to generate a professional README.
